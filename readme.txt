## Installation
* Open Terminal (ctrl+Alt+T)
* Git clone-https:https://github.com/Nobella-Nyarari-Ejiofor/
* Git fork-Enter your own repository and search https://github.com/Nobella-Nyarari-Ejiofor/ then click on fork to add
it on your own repository.
